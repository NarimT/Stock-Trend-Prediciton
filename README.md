# AT82.03-Stock-Trend-Predictor

How to install development environment:
1. Install uv
2. Install TA-Lib
  - https://ta-lib.org/install/
  - For Windows, download executable and install normally
  - For Mac, read instructions on the website
  - This is C++ dependency for python wrapper, otherwise Ta-Lib python package will not work
3. Clone this repository
4. cd into the repository and run `uv sync`
5. Hope it works :)
